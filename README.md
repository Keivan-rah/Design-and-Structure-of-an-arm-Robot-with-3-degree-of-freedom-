# Design-and-Structure-of-an-arm-Robot-with-3-degree-of-freedom-
After design and structure of the robot, implemented Visual Servoing for equipping the Robot with Machine Vision.
This project, titled Visual Servoing, consists of two parts: image processing with Python and sending it to a robotic arm (Robot End-Effector).
Every color image is actually a two-dimensional matrix of numbers that can represent the color, depth,
brightness, or other characteristics of the image. This means that we can see the image by presenting
these matrices to the display. These matrices have different standards for color or intensity arrangement. For example, HSV or RGB. Therefore, specific mathematical operations can be performed on this matrix of numbers, which can lead to
improving the quality of the image, changing the color of the image, or adding or subtracting something to
or from the image, or recognizing the desired image for a specific operation. This process is called image
processing.
